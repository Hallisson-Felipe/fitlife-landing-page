# FitLife Landing Page

Uma landing page limpa, branca e sofisticada desenvolvida para apresentar o aplicativo fitness **FitLife**. O design foi concebido com uma estética premium e minimalista, focando no destaque dos prints do aplicativo e em uma navegação fluida.

---

## 🚀 Funcionalidades da Página

- **Header Responsivo**: Cabeçalho flutuante (sticky) com efeito de desfoque (glassmorphism), logotipo personalizado em CSS e botão de ação rápido.
- **Seção Hero**: Com altura de `90vh`, estruturada em 3 áreas horizontais:
  - **Esquerda**: Headline elegante e direta.
  - **Centro**: Mockup do dispositivo móvel contendo a tela principal do app em destaque.
  - **Direita**: Lista elegante de tópicos essenciais (Saúde, Exercício, Amigos).
- **Seção de Funcionalidades**: Exposição alternada e dinâmica (imagem/texto e texto/imagem) dos recursos do app:
  - Contagem de Calorias e Macronutrientes.
  - Grade de Exercícios Específicos (Corrida, Musculação, etc.).
  - Integração social e Compartilhamento com Amigos.
- **Chamada para Ação (CTA)**: Área destacada para download do app com o selo "Totalmente gratuito".
- **Rodapé Clean**: Declaração simples de direitos autorais.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando apenas tecnologias web fundamentais, priorizando técnicas modernas, simplicidade de código e facilidade de manutenção:

- **HTML5 Semântico**: Uso de tags semânticas (`<header>`, `<main>`, `<section>`, `<footer>`) para melhor acessibilidade e SEO.
- **CSS3 Moderno**:
  - Organização e alinhamento de elementos estruturados inteiramente com **Flexbox**.
  - **Custom Properties (Variáveis CSS)** para definição de uma paleta de cores consistente e reutilizável.
  - Transições suaves (`transition`) e animações micro-interativas nos botões no estado de hover.
  - **Responsividade total** feita de forma defensiva com CSS Media Queries para se adaptar perfeitamente a computadores, tablets e smartphones (com ajustes adicionais para telas pequenas abaixo de 480px).
- **Fontes do Google**: 
  - *Roboto*: Utilizada como fonte principal para leitura limpa e moderna.
  - *Emblema One*: Utilizada especificamente para a identidade visual do logotipo.
